  LAYER 5 — ENSEMBLE VOTING  (every cycle)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  6 agent signals arrive per symbol (GeminiAgent excluded from vote):

  ┌─────────────────────────────────────────────────────────────────────┐
  │                                                                     │
  │  TechAgent ──────────── weight × confidence ─────────┐            │
  │  MomentumAgent ───────── weight × confidence ─────────┤            │
  │  MeanReversionAgent ──── weight × confidence ─────────┤  Σ        │
  │  SentimentAgent ─────── weight × confidence ─────────┤            │
  │  HistoricalTrendsAgent ── weight × confidence ────────┤            │
  │  ClaudeAgent ────────── weight × confidence ─────────┘            │
  │                                                                     │
  │  buy_score  = Σ(weight × conf) for BUY  signals / total_weight    │
  │  sell_score = Σ(weight × conf) for SELL signals / total_weight    │
  │                                                                     │
  └─────────────────────────────────────────────────────────────────────┘

  Every 5 cycles — weights are recalculated:

  ┌──────────────────────────────────────────────────────────────────────┐
  │  STEP 1 — Performance scoring                                        │
  │                                                                      │
  │  perf_score = 0.5 × win_rate + 0.5 × norm_sharpe                   │
  │  norm_sharpe = (sharpe + 2) / 5    [clamped -2 to +3 → 0 to 1]     │
  │  (if < 5 trades → perf_score = 1.0, neutral)                        │
  │                                                                      │
  │  STEP 2 — Regime detection  (SPY/QQQ bars, 2-of-3 signals required) │
  │                                                                      │
  │  Signal A: abs(SMA20 slope) > 0.4%  per price                       │
  │  Signal B: >60% of last 10 days in slope direction                  │
  │  Signal C: recent 5-bar volume > 20-bar avg × 1.10                 │
  │                                                                      │
  │  2+ of A/B/C true  →  TRENDING                                      │
  │  ATR > 2.5% of price  →  VOLATILE   (checked first)                 │
  │  otherwise  →  RANGING                                              │
  │                                                                      │
  │  STEP 3 — Build final weights                                        │
  │                                                                      │
  │  raw = base_weight × max(0.30, perf_score) × regime_multiplier      │
  │  final = raw / Σ(all raw)                                            │
  └──────────────────────────────────────────────────────────────────────┘

  Regime multipliers:

  Agent                   TRENDING    RANGING    VOLATILE
  ──────────────────────  ─────────   ────────   ────────
  MomentumAgent             ×1.50      ×0.55      ×0.65
  TechAgent                 ×1.20      ×1.10      ×0.80
  HistoricalTrendsAgent     ×1.20      ×1.30      ×0.80
  ClaudeAgent               ×1.10      ×1.00      ×1.40
  SentimentAgent            ×0.80      ×1.20      ×1.20
  MeanReversionAgent        ×0.40      ×1.60      ×0.60

  Base weights (before adjustment):
  ClaudeAgent 29% · TechAgent 23% · SentimentAgent 17%
  MomentumAgent 14% · MeanReversionAgent 9% · HistoricalTrendsAgent 8%

  ┌─────────────────────────────────────────────────────────────────────┐
  │  FINAL VOTE DECISION                                                │
  │                                                                     │
  │  buy_score  >= 0.60  AND  no existing position                      │
  │           AND  buy_score >= sell_score × 2.0 (margin of safety)    │
  │                                             →  BUY                 │
  │  sell_score >= 0.60  AND  holding position  →  SELL                │
  │  neither clears threshold, or MoS fails     →  HOLD                │
  │                                                                     │
  │  Position size:                                                     │
  │  target = portfolio_value × MAX_POSITION_SIZE × buy_score          │
  │  target = min(target, available_cash × 0.95)                       │
  │  shares = floor(target / price × 100) / 100                        │
  └─────────────────────────────────────────────────────────────────────┘

  FULL CYCLE SUMMARY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [Every 60s, 9:30–16:00 ET]

  1. Fetch market data (Alpaca + Stooq long-term) + build market_context
  2. (Every 3rd cycle) GeminiAgent → market_analysis injected into context
  3. Run 6 agents in parallel → each returns BUY/SELL/HOLD per symbol
     (TechAgent, MomentumAgent, MeanReversionAgent, SentimentAgent,
      HistoricalTrendsAgent, ClaudeAgent)
     └─ OLLAMA_ONLY_MODE=1: ClaudeAgent prepends Phase 1 few-shot
        examples from learning.json before sending to llama3.1:8b
  4. EnsembleAgent collects all signals → weighted vote per symbol
     └─ Every 5 cycles: recalculate regime + adaptive weights
     └─ BUY gate: threshold 0.60 AND margin-of-safety 2× vs sell signal
  5. For each signal: RiskManager gate
