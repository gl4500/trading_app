TRADING APP — AGENT BUY/SELL FORMULAS

--------------------------------------------------------------------------------
TECHAGENT — Rule-based composite scoring
--------------------------------------------------------------------------------
Indicators: RSI (14-period), MACD (12/26/9), Bollinger Bands (20-period, 2σ),
            Volume SMA-20

BUY score (threshold >= 0.55, no existing position):
  RSI < RSI_OVERSOLD (default 35)                          +0.35
  Price <= lower Bollinger Band                             +0.30
  MACD bullish crossover (histogram flips negative to 0)   +0.25
  MACD histogram just positive (no cross)                  +0.10
  Volume > 1.2x 20-day avg                                 +0.10

SELL score (threshold >= 0.55, must hold position):
  RSI > RSI_OVERBOUGHT (default 65)                        +0.35
  Price >= upper Bollinger Band                            +0.30
  MACD bearish crossover                                   +0.25
  MACD histogram just negative                             +0.10
  Volume spike confirmation                                +0.10

Position sizing:
  shares = floor((portfolio_value x MAX_POSITION_SIZE x buy_score) / price x 100) / 100


--------------------------------------------------------------------------------
MOMENTUMAGENT — Rate-of-change + trailing stop
--------------------------------------------------------------------------------
BUY score (threshold >= 0.30, must also have positive VW-momentum):
  5D momentum > 2% threshold                 min(0.30, mom_5d x 5)
  10D momentum > 1%                          min(0.25, mom_10d x 3)
  20D momentum > 0                           min(0.15, mom_20d x 2)
  Trend consistency > 60% positive days      +0.15
  Momentum accelerating (5D accel > 0)       +0.10
  Recent volume > 1.2x historical avg        +0.05

SELL conditions (threshold >= 0.30 OR 5D momentum < -2%):
  5D momentum < -2%                          min(0.40, abs(mom) x 5)
  10D momentum reversal < -1%                min(0.30, abs(mom) x 3)
  Momentum decelerating (accel < -0.01)      +0.15
  Trend consistency < 40%                    +0.15

Trailing stop (instant SELL, confidence=0.90):
  (high_water_mark - price) / high_water_mark > TRAILING_STOP (default 3%)

Moving average trend:
  ma_trend = (SMA5 - SMA20) / SMA20

Volume-weighted momentum:
  vol_weights = volume / volume.sum()
  vw_momentum = sum(recent_returns x vol_weights) over last 5 days


--------------------------------------------------------------------------------
MEANREVERSIONAGENT — Z-score
--------------------------------------------------------------------------------
Core formula:
  z = (current_price - rolling_mean_20) / rolling_std_20

BUY: z <= -1.5 and no existing position
  confidence = min(0.95, |z| / 3.0)
  Bonus +0.10 if z-score is already trending back toward 0

SELL (overbought): z >= +1.5 and holding position
  confidence = min(0.95, |z| / 3.0)
  Bonus +0.10 if z-score is already trending back toward 0

SELL (profit take): holding, -0.3 <= z <= 0.5 (mean reversion complete), PnL > 1%
  confidence = 0.70

Position sizing:
  z_factor     = min(1.0, |z| / 3.0)
  target_alloc = portfolio_value x MAX_POSITION_SIZE x z_factor


--------------------------------------------------------------------------------
SENTIMENTAGENT — GPT-4o-mini LLM
--------------------------------------------------------------------------------
Sends price action description to GPT-4o-mini:
  - 1D / 5D / 20D returns
  - Volume ratio (recent 3-day avg / 20-day avg)
  - Distance from 20D high and low
  - Daily price volatility (std of last 10 days)
  - Consecutive up/down days (last 5)

LLM returns: { sentiment, confidence, strength }

Strength multiplier: strong=1.0, moderate=0.8, weak=0.6
adjusted_confidence = confidence x strength_multiplier

BUY:  sentiment == "bullish" AND adjusted_confidence >= 0.45, no existing position
SELL: sentiment == "bearish" AND adjusted_confidence >= 0.45, must hold position

Throttle: one batch (up to 5 symbols) every 90 min during market hours,
          every 4 hours off-hours. Daily token cap: 10,000.


--------------------------------------------------------------------------------
CLAUDEAGENT — Claude Opus 4.6 LLM
--------------------------------------------------------------------------------
Receives full market context per symbol:
  - Current price, 1D/5D/20D % change, 52W high/low
  - RSI, MACD, Bollinger Bands, SMA20
  - Multi-source composite signal (analyst 35%, earnings 22%, news 18%,
    Yahoo news 12%, congressional trades 13%)
  - Alpaca news headlines (last 24h)
  - OHLCV bars (last 30 days)
  - Portfolio state (cash, positions, P&L)
  - Learning history (past trade outcomes)
  - Overnight/after-hours catalysts
  - Gemini's independent market view

Decision framework (instructed in prompt):
  STRONG BUY:  Bullish news catalyst + RSI < 65 + MACD positive/crossing
               + price > SMA20
  STRONG SELL: Negative news + RSI > 65 + MACD negative/crossing
               + price < SMA20
  CONFLICTED:  News vs. technicals disagree — use technicals for timing,
               news for direction

Output: free-form confidence (0.0-1.0), only buys with strong conviction.
Rate limit: 2 API calls per hour (sliding window).


--------------------------------------------------------------------------------
GEMINIAGENT — Gemini 2.0 Flash LLM  [NEWS-ONLY — excluded from ensemble vote]
--------------------------------------------------------------------------------
Receives same market data as ClaudeAgent (minus learning history).
Runs every 3rd cycle as a pre-pass; its market_analysis string is injected
into ClaudeAgent's prompt as "## Gemini Market View".

Decision framework (instructed in prompt):
  STRONG BUY:  Bullish catalyst + RSI < 65 + positive MACD + price > SMA20
  STRONG SELL: Negative catalyst + RSI > 65 + negative MACD + price < SMA20
  HOLD:        Mixed signals or already at target allocation

GeminiAgent produces its own BUY/SELL/HOLD signals for display purposes but
does NOT participate in EnsembleAgent voting — it functions as a research
assistant that informs ClaudeAgent.
Rate limit: 2 API calls per hour (sliding window).


--------------------------------------------------------------------------------
HISTORICALTRENDSAGENT — Seasonal + channel + multi-period momentum
--------------------------------------------------------------------------------
Four sub-scores combined into a composite:
  composite = seasonal×0.20 + channel×0.30 + momentum×0.40 + volume×0.10

1. Seasonal score (month-of-year bias, research-based):
   Jan +0.40 · Feb +0.10 · Mar +0.10 · Apr +0.20 · May -0.20 · Jun -0.15
   Jul +0.05 · Aug -0.15 · Sep -0.30 · Oct +0.10 · Nov +0.35 · Dec +0.40
   Plus quarter-position bias:
     last ~6 weeks of quarter (window dressing)  +0.15
     first 2 weeks of new quarter                +0.10
   Combined: month_bias×0.70 + quarter_bias×0.30

2. Channel analysis (price position within historical high-low range):
   position = (price - period_low) / (period_high - period_low)
   raw      = (0.5 - position) × 2.0      (near low=+1, near high=−1)
   adjusted = raw + SMA_slope × 5.0       (uptrend moderates bearish reading)
   score clamped to [-1, +1]

3. Multi-period momentum (5/10/20/40-day windows):
   weights: 5d=15%, 10d=20%, 20d=30%, 40d=35%
   weighted_roc = sum(roc × weight) / total_weight
   score = weighted_roc × 5.0
   Alignment bonus:  >75% timeframes agree → +0.20
   Alignment penalty: <25% timeframes agree → -0.20

4. Volume trend confirmation (small signal ±0.15):
   vol_ratio = (up_day_vol - down_day_vol) / (up_day_vol + down_day_vol)
   score = vol_ratio × 0.15

BUY:  composite > +0.25 and no existing position
SELL: composite < -0.25 and holding position

Confidence = min(0.95, max(0.05, |composite|))

Data source: prefers Stooq 5-year bars (STOOQ_LONG_TERM_DAYS=1250),
             falls back to 60-day Alpaca bars.
Minimum 30 bars required.

Position sizing:
  target_alloc = portfolio_value × MAX_POSITION_SIZE × confidence
  target_alloc = min(target_alloc, cash × 0.95)
  shares = floor(target_alloc / price × 100) / 100


--------------------------------------------------------------------------------
ENSEMBLEAGENT — Adaptive weighted voting
--------------------------------------------------------------------------------
Aggregates signals from TechAgent, MomentumAgent, MeanReversionAgent,
SentimentAgent, HistoricalTrendsAgent, and ClaudeAgent.
GeminiAgent is excluded from the vote (news-only role).

Vote scoring:
  buy_score  = sum(weight x confidence) for BUY  signals / total_weight
  sell_score = sum(weight x confidence) for SELL signals / total_weight

Decision:
  BUY  if buy_score  >= ENSEMBLE_THRESHOLD (0.60)
       AND no existing position
       AND buy_score >= sell_score × MARGIN_OF_SAFETY (2.0)
  SELL if sell_score >= ENSEMBLE_THRESHOLD (0.60), must hold position
  HOLD if neither condition met, or margin-of-safety fails

Adaptive weights (recomputed every 5 cycles):
  perf_score   = 0.5 x win_rate + 0.5 x norm_sharpe
  norm_sharpe  = clamp(sharpe, -2, +3) -> [0,1]  = (sharpe + 2) / 5
  raw_weight   = base_weight x max(0.30, perf_score) x regime_multiplier
  final_weight = raw_weight / sum(all raw_weights)

Base weights:
  ClaudeAgent:           29%
  TechAgent:             23%
  SentimentAgent:        17%
  MomentumAgent:         14%
  MeanReversionAgent:     9%
  HistoricalTrendsAgent:  8%

Regime detection (every 5 cycles, using SPY/QQQ bars):
  Volatile  (ATR > 2.5%):
    Claude ×1.40, Sentiment ×1.20, HistTrends ×0.80, Tech ×0.80,
    Momentum ×0.65, MeanRev ×0.60
  Trending  (2-of-3: SMA slope, consistency >60%, volume expansion):
    Momentum ×1.50, Tech ×1.20, HistTrends ×1.20, Claude ×1.10,
    Sentiment ×0.80, MeanRev ×0.40
  Ranging   (default):
    MeanRev ×1.60, HistTrends ×1.30, Sentiment ×1.20, Tech ×1.10,
    Claude ×1.00, Momentum ×0.55

Regime signals:
  SMA slope  = (SMA20[-1] - SMA20[-10]) / current_price  (threshold abs > 0.004)
  ATR        = mean(high - low, last 14 bars) / current_price
  Consistency = pct of last 10 days returns in same direction as slope (>60%)
  Volume     = recent 5-bar avg > 20-bar avg x 1.10


--------------------------------------------------------------------------------
SCANNERPORTFOLIOAGENT — Claude scanner picks
--------------------------------------------------------------------------------
Acts on cached scanner recommendations (no independent price formula).

BUY:  scanner action == BUY AND confidence >= 0.60
      Shares sized by RiskManager.get_max_buy_shares()
      Only enters new positions on a fresh scan (avoids churning same scan)

SELL (explicit):    scanner action == SELL for a held position
SELL (stop-loss):   held position NOT in current BUY set
                    AND unrealized loss >= DEFAULT_STOP_LOSS_PCT (5%)
