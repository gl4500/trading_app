  cycle_count % 15 == 1 ?
       │
       ├── YES → await get_macro_context_text()
       │              │
       │              ├── Cache still valid? (< 15 min old)
       │              │        │
       │              │        └── YES → return cached text (no yfinance call)
       │              │
       │              └── NO  → fetch fresh data (yfinance, executor thread)
       │                             │
       │                             ├── Data returned? ──── NO → return stale
       │                             │                             cache text
       │                             └── YES → build new text, update cache
       │
       └── NO  → carry forward __macro_context__ from previous cycle
                 (prevents missing key if never fetched yet)

  LAYER 2 — 6 INTER-MARKET RULES  (_apply_intermarket_rules)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Each rule emits:  signal string  +  sector bias tags (bullish / bearish)
  Bias tags use setdefault() — first rule to tag a sector wins.

  ─────────────────────────────────────────────────────────────────────────────
  RULE 1 — GOLD  (Murphy: lead indicator for risk-off)
  ─────────────────────────────────────────────────────────────────────────────

  GLD 5-day return?
  │
  ├── > +2.0%  →  RISK-OFF signal
  │             XLU, XLV, XLP → BULLISH   (defensives benefit)
  │             XLK, XLY, XLRE → BEARISH  (growth/rate-sensitive hit)
  │
  ├── +0.8% to +2.0%  →  MILD RISK-OFF signal (monitor only, no bias)
  │
  ├── -1.5% to +0.8%  →  No signal
  │
  └── < -1.5%  →  RISK-ON signal
                  XLK, XLY, XLI, XLF → BULLISH  (growth/cyclicals favoured)
                  XLU, XLP → BEARISH             (defensives fade)

  ─────────────────────────────────────────────────────────────────────────────
  RULE 2 — BONDS / YIELDS  (Murphy: bond market leads stocks)
  ─────────────────────────────────────────────────────────────────────────────

  TLT 5-day return?   (TLT price FALLS when yields RISE)
  │
  ├── < -2.0%  →  YIELDS RISING SHARPLY
  │             XLF → BULLISH              (wider net interest margins)
  │             XLU, XLRE, XLK → BEARISH  (higher discount rate hurts)
  │
  ├── -2.0% to -0.8%  →  Yields drifting higher
  │             XLU, XLRE → BEARISH (mild)
  │
  ├── -0.8% to +2.0%  →  No signal
  │
  └── > +2.0%  →  YIELDS FALLING (flight to safety)
                  XLU, XLRE, XLV → BULLISH  (rate-sensitives benefit)
                  Signal: growth concern — XLK, XLY may lag

  ─────────────────────────────────────────────────────────────────────────────
  RULE 3 — US DOLLAR  (Murphy: inverse commodities + EM pressure)
  ─────────────────────────────────────────────────────────────────────────────

  UUP 5-day return?
  │
  ├── > +1.5%  →  STRONG DOLLAR
  │             XLB, XLE → BEARISH  (commodity headwind + FX drag)
  │
  ├── -1.5% to +1.5%  →  No signal
  │
  └── < -1.5%  →  WEAK DOLLAR
                  XLB, XLE → BULLISH  (commodity tailwind)

  ─────────────────────────────────────────────────────────────────────────────
  RULE 4 — OIL  (energy cost pass-through across sectors)
  ─────────────────────────────────────────────────────────────────────────────

  USO 5-day return?
  │
  ├── > +4.0%  →  ENERGY SURGE
  │             XLE → BULLISH           (direct beneficiary)
  │             XLY, XLI → BEARISH     (higher input costs)
  │
  ├── -4.0% to +4.0%  →  No signal
  │
  └── < -4.0%  →  OIL FALLING
                  XLE → BEARISH         (revenue hit)
                  XLY → BULLISH         (consumer relief)

  ─────────────────────────────────────────────────────────────────────────────
  RULE 5 — VIX / FEAR INDEX
  ─────────────────────────────────────────────────────────────────────────────

  VIX level?
  │
  ├── > 30  →  EXTREME FEAR
  │            XLU, XLV, XLP → BULLISH  (defensives override all)
  │            XLK, XLY, XLI → BEARISH  (high-beta at max risk)
  │            Signal: consider reducing all equity exposure
  │
  ├── 20–30  →  ELEVATED FEAR
  │             Signal only — prefer lower-beta, no bias overrides
  │
  ├── 14–20  →  Normal range — no signal
  │
  └── < 14   →  LOW FEAR / COMPLACENCY
                Signal: risk-on environment supports cyclicals + growth

  ─────────────────────────────────────────────────────────────────────────────
  RULE 6 — BRIDGEWATER ALL WEATHER REGIME  (growth/inflation matrix)
  ─────────────────────────────────────────────────────────────────────────────

  Evaluated in priority order (first match wins signal):

  STAGFLATION RISK?    GLD>+1.5% AND USO>+3.0% AND TLT<-1.0%
       │
       └── YES → Signal: STAGFLATION
                 Hard assets (XLE, XLB) > Defensives > Growth

  INFLATION REGIME?    GLD>+1.0% AND USO>+2.0% AND TLT<-0.5%
       │
       └── YES → Signal: INFLATION REGIME
                 Overweight: XLE, XLB   Underweight: XLK, bonds

  DEFLATION / RECESSION?    GLD>+1.0% AND TLT>+1.5% AND VIX>20
       │
       └── YES → Signal: DEFLATION / RECESSION FEAR
                 Overweight: XLU, XLV, XLP, TLT   Underweight: XLE, XLB, XLY

  GROWTH REGIME?    GLD<-0.5% AND TLT<-0.5% AND USO<+1.0%
       │
       └── YES → Signal: GROWTH REGIME
                 Overweight: XLK, XLF, XLY, XLI   Underweight: TLT, XLU

  (none match)  →  No Bridgewater signal this cycle

  LAYER 3 — OVERALL REGIME CLASSIFICATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  After all 6 rules run, count bullish/bearish sector tags.
  Evaluated in priority order (first match wins):

  stagflation_risk  (Rule 6 flag)         →  "STAGFLATION RISK"
  deflation_signal  OR  GLD_5d > +2.0%    →  "RISK-OFF / DEFENSIVE"
  growth_signal     AND  VIX < 18         →  "RISK-ON / GROWTH"
  inflation_signal  (Rule 6 flag)         →  "INFLATIONARY"
  VIX > 25                                →  "HIGH VOLATILITY"
  bearish_count > bullish_count + 2       →  "CAUTIOUS"
  bullish_count > bearish_count + 2       →  "CONSTRUCTIVE"
  (default)                               →  "NEUTRAL / MIXED"

  │    ...                                                                  │
  │                                                                         │
  │  TRADING IMPLICATIONS FOR YOUR WATCHLIST:                               │

  Momentum arrows:
    ↑↑  =  5d > +2.0% AND 20d > +3.0%  (strong uptrend)
    ↑   =  5d > +0.5%
    →   =  flat / insufficient data
    ↓   =  5d < -0.5%
    ↓↓  =  5d < -2.0% AND 20d < -3.0%  (strong downtrend)

  HOW AI AGENTS USE MACRO CONTEXT:

  Stock on watchlist is in a BEARISH sector?
       │
       ├── BUY signal from technicals?
       │        → Require stronger conviction (higher conf threshold)
       │        → Ollama/Claude instructed: "require stronger conviction
       │            in macro headwind sectors before buying"
       │
       └── SELL signal from technicals?
                → Macro headwind confirms — execute with normal confidence

  Stock on watchlist is in a BULLISH sector?
       │
       ├── BUY signal from technicals?
       │        → Macro tailwind confirms — apply as tiebreaker
       │
       └── Mixed technicals?
                → Macro tailwind can tip the balance toward BUY
